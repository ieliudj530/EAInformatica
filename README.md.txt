# EAInformática Gestión

**Sistema de Gestión de Clientes, Ventas e Inventario**

Bienvenido a **EAInformática Gestión**, una herramienta web diseñada para pequeñas empresas que desean llevar un control eficiente de clientes, ventas, productos y cuentas por cobrar, utilizando Google Sheets como base de datos.

## 📦 Características Principales
- Registro de clientes con CPF/CNPJ.
- Gestión de ventas y pagos con cálculo automático de deuda.
- Inventario de productos con control de stock.
- Listado de cuentas por cobrar, agrupadas por cliente.
- Interfaz amigable y adaptable a dispositivos móviles.

## 🛠️ Requisitos Técnicos
- Navegador actualizado (Google Chrome recomendado).
- Conexión a internet.
- Cuenta de Google con acceso a Google Sheets.
- Apps Script habilitado y vinculado al HTML.

## 🚀 Guía Rápida de Uso

### 1. Registrar Cliente
- Ingresa nombre, CPF/CNPJ, email, teléfono, dirección y fecha.
- Haz clic en **Guardar Cliente**. Los datos se almacenan en la hoja `Clientes`.

### 2. Buscar Cliente y Registrar Venta
- Introduce CPF/CNPJ en el buscador.
- Verás el historial de ventas y podrás:
  - Registrar nueva venta.
  - Abonar pagos pendientes.
  - Visualizar productos vendidos por venta.

### 3. Inventario
- Visualiza productos disponibles, precio y stock.
- Agrega productos nuevos o actualiza stock.
- Edita o elimina productos desde el menú `...`.

### 4. Cuentas por Cobrar
- Visualiza deuda total por cliente.
- Despliega detalles de cada venta pendiente.

## ❓ Preguntas Frecuentes
- **¿Dónde se guarda la información?**  
  En hojas de cálculo de Google Sheets vinculadas mediante Apps Script.

- **¿Qué pasa si ingreso mal un dato?**  
  Puedes editar directamente desde Google Sheets o mediante las funciones del sistema.

- **¿Funciona en celular?**  
  Sí, la interfaz es responsive y compatible con móviles.
